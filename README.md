# 传奇掉落查询工具

## 项目简介

这是一个用于解析和查询传奇游戏爆率文件的桌面应用程序。支持标准的传奇爆率文件格式，包括`#CHILD`结构。

## 功能特性

- ✅ 解析标准的传奇爆率文件格式
- ✅ 支持`#CHILD 1/10 RANDOM`子爆率结构
- ✅ 物品搜索和筛选功能
- ✅ 怪物掉落列表查看
- ✅ 详细的爆率信息显示
- ✅ 数据导出为CSV格式
- ✅ 跨平台支持（Windows/macOS/Linux）

## 安装方法

### 方法一：使用预编译版本

1. 从 Releases 页面下载最新版本的 `LegendDropTool.zip`
2. 解压到任意目录
3. 运行 `LegendDropTool.exe`

### 方法二：从源码运行

1. 确保已安装 Python 3.8 或更高版本
2. 克隆或下载本项目
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
